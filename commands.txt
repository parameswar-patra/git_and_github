____________________________________________________________________________________________________________________________________________________________
git branch                            =to check the List all branches, The current branch is marked with *
git branch <new branch name>          =to create a new branch
git checkout <branch name>            =to checkout to another branch(Alternative=git switch <branch name>)
git checkout -b <new branch name>     =to create a new branch and checkout to that new branch(Alternative=git switch -c <branch name>)
git branch -m <new branch name>       =to rename an existing branch
git branch -d <branch name>           =to delete an existing branch
git merge <branch name>               =merge
git diff <branch name>                =to check the difference between current branch and the specified branch
